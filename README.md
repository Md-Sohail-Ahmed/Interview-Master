# Interview-Master
Setup Instructions

Clone the repository

git clone https://github.com/your-username/Interview-Master.git
cd Interview-Master
Setup Backend
cd backend
npm install

Create a .env file:

GOOGLE_API_KEY=your_api_key_here
MONGO_URI=your_mongodb_connection
PORT=3000

Run backend:

npm run dev
 Setup Frontend
cd frontend
npm install
npm run dev
Environment Variables

Create a .env file in the backend folder using:

GOOGLE_API_KEY=your_api_key
MONGO_URI=your_db_url
PORT=3000
